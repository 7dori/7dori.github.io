Workshop 1: Learning the Basics

“Take the first step in faith. You don’t have to see the whole staircase, just take the first step.”
– Martin Luther King Jr.
Monday, Sept 7

Class Description

In class one we’ll be building your first ‘Hello World!’ and ‘Cat appreciation Page’. We’ll work on File Management, Download a Text-Editor (VS Code) and download packages for faster development, and familiarize ourselves with basic tags. This intro class will be a presentation, followed up by a quick demo of the topics introduced.
Check out the Lecture Slides Here.


Outcomes

By the end of the first workshop we should have set-up a Helloworld file, Downloaded a Text-editor, explored various Tags, gone through the presentation slides, and demo'd what we'll be working on next week.
